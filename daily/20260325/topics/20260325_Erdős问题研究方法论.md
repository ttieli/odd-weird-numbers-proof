# Erdős 问题研究方法论 — 学习不被束缚

- 来源：2022-2025 多篇学术论文 + OEIS + 实验
- 日期：2026-03-25
- 聚焦问题：奇数奇怪数 ($1,000) + R(5,5) ($250)

## 一、学到的方法（可借鉴）

### 1. 反向搜索（Reverse Search）— 最重要的突破

**来源**：arXiv:2207.12906（2022, Hasegawa & Melfi 等）

传统方法：逐个检查每个奇数 → 太慢
反向搜索：**从因子分解结构出发，自上而下生成候选**

原理：
- 不搜"所有奇数"，搜"所有可能的素因子组合"
- 奇怪数 = 丰数 + 不半完全
- 先构造所有"恰好是丰数"的素因子组合（边界丰度）
- 再检查这些候选是否半完全
- 搜索空间从 O(N) 降到 O(素因子组合数)

**跨领域来源**：多面体顶点枚举算法（组合优化 → 数论）

### 2. 边界丰度策略（Boundary of Abundance）

关键洞察：**过剩量越小，越有可能是奇怪数**

- 偶数最小奇怪数 70：过剩量仅 4（74-70）
- 所有已知偶数奇怪数的过剩量都很小
- 所以搜奇数奇怪数时，优先搜 σ(n)-n 最小的丰数

**实际效果**：搜到 10^28 对过剩量 < 10^14 的情况，比完整搜索快数万倍

### 3. 素因子参数化（Factorization-based Enumeration）

**来源**：Amato, Hasler, Melfi, Parton (2019, JNT)

不搜"所有数"，搜"所有有可能的素因子幂次组合"：
- 固定 Ω(n) = k（素因子个数含重复）
- 枚举所有满足 σ(n)/n > 2 的 (p1^a1, p2^a2, ...) 组合
- 用丰度公式 σ(n)/n = Π σ(pi^ai)/pi^ai 直接筛选

**关键公式**：
```
σ(p^a)/p^a = (p^(a+1)-1) / (p^a(p-1))
```
每个素因子的"丰度贡献"可以独立计算！

### 4. 子集和的贪心优化

检查半完全性（子集和 = n）是 NP 完全的，但实际中：
- 从最大因子开始贪心
- 大因子减少后搜索空间急剧缩小
- 配合动态规划（bitset），50万以内 < 1秒

**更快的方法**：meet-in-the-middle（分半搜索）
- 因子分两半，各自枚举子集和
- 查找两半是否有和 = n 的组合
- 复杂度从 O(2^d) 降到 O(2^(d/2))

### 5. SAT/SMT 求解器（跨领域借鉴）

**来源**：R(5,5) 研究 + Schur 数 + 布尔 Pythagoras 三元组

把数论条件编码为布尔公式：
- 变量：n 的每个因子"是否选入子集"
- 约束：选入因子之和 = n

如果 SAT 求解器判定 UNSAT → n 不是半完全数 → 可能是奇怪数

**工具**：CaDiCaL、Kissat（最先进的 CDCL 求解器）

## 三、可能的新进攻路线

### 路线 A：利用"不被 3 整除"的线索

**观察**：没有已知奇怪数被 3 整除，而所有奇数丰数被 3 整除。

如果能证明"被 3 整除的丰数一定是半完全数" → 自动证明无奇数奇怪数。

为什么有戏？
- 3|n → n/3 是一个因子，且 n/3 ≈ 0.33n（大因子作为"锚点"）
- n/3 + n/9 = 4n/9 ≈ 0.44n
- n/3 + n/9 + n/27 → 趋向 n/2（几何级数）
- 3 的幂次因子本身就能覆盖 [n/3, n/2] 区间
- 配合其他因子（5, 7, ...），可能能证明完全覆盖 [0, n]

### 路线 B：丰度指数下界

证明：如果 n 是奇数丰数，则 d(n) ≥ f(σ(n)-n)，使得 2^d(n) >> σ(n)-n

即：因子数增长速度永远压制过剩量 → 子集和一定能覆盖

### 路线 C：模运算覆盖定理

利用奇数丰数的因子间整除关系，证明子集和在 mod 小数 下能覆盖所有余数类 → 然后组合成覆盖 [0, n]

### 路线 D：与其他猜想的关联

- **Cramér 猜想**（素数间隔）→ Melfi 已证明可推出原始奇怪数无穷多
- **奇完全数不存在猜想** → 如果奇完全数不存在，是否可以推出奇数半完全数"总是存在"？
- **Goldbach 猜想**的子集和视角
