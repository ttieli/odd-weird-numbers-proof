## 项目记忆

### 对话开始时

1. 读取 `.claude/project-memory.md`（如存在）
2. 扫描 `.claude/sessions/*.md`（未合并的会话）
3. 按时间排序，重点关注最新记录
4. 简要告知用户已了解的上下文（一句话即可）
5. 归档子文件（`.claude/archive/*.md`）通常不读，除非用户明确要查历史

### 对话结束时

当用户说"再见"、"bye"、"结束"、"保存进度"时：
- 使用 `/project-memory` skill 保存会话

### 自动保存

满足以下任一条件时，自动调用 `/project-memory` 保存当前进度（静默执行，保存后简短提示一句）：
- 完成了一个明确的功能或修复
- 对话轮数达到 20 轮
- 用户开始一个新的、不相关的任务
- 收到 context 压缩警告时（优先于压缩执行）

### 其他命令

| 用户说 | 动作 |
|--------|------|
| 合并记忆 / merge sessions | 合并所有 session 到主记忆文件 |
| 查看会话 / list sessions | 列出所有未合并的会话 |
